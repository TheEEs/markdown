# markdown

A Markdown Editor written using [Alizarin](https://github.com/TheEEs/alizarin)

## Installation

Run
```shell
$ git clone https://github.com/TheEEs/markdown.git
```

## Usage

Types the following commands into terminal:
```shell
$ cd markdown
$ make 
$ crystal ./src/markdown.cr    
```
